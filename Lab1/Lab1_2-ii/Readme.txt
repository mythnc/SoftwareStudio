step:
1. 使用者輸入字串
同2-i

2. 判斷字串是否為回文
同2-i

3. 判斷是否能增加一個字元，使字串變為回文
利用for loop，讓i從index 0往後走，j從index len - 1往前走。
當兩者的值不相等時比對i + 1與j是否相等。
若相等時，紀錄j + 1的位置。
若不相等，比對j - 1與i是否相等，
若相等時，紀錄i的位置。
又不相等時，可以知道無法讓字串變為回文，迴圈終止。

利用變數pos紀錄位置，pos初始為-1。
當pos不為-1時，表示已有紀錄值，此時若i與j值又不相等，
表示無法只增加一個字元，讓字串變為回文。

利用變數c紀錄欲增加的字元值。

4. 輸出判斷結果
為回文輸出「this is a palindrome.」
不為回文輸出「no result.」
空字串輸出「you must input one character at least.」
可增加一個字元，使字串變為回文時，輸出該字元，與新的回文字串。

遇到的困難：
char c需設初始值，否則compile不會過

解決方法：
乖乖設定初始值
